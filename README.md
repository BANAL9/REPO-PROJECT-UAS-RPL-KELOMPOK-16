## üöÄ Fitur Utama
* **Sistem Login**: Autentikasi keamanan menggunakan MD5.
* **Dashboard Statistik**: Ringkasan jumlah menu dan total transaksi.
* **CRUD Menu**: Manajemen data makanan/minuman (Nama, Harga, Stok, Kategori).
* **Sistem Transaksi**: Pencatatan pesanan pelanggan dan nomor meja.
* **Kasir Real-time**: Perhitungan pembayaran dan uang kembali otomatis dengan JavaScript.
* **Laporan Pendapatan**: Rekapitulasi transaksi lunas dengan fitur cetak laporan.

## üìã Metodologi Pengembangan
Proyek ini menggunakan metode **Agile Development (Scrum)** karena:
1.  **Dinamis**: Memudahkan penyesuaian fitur sesuai kebutuhan operasional kantin.
2.  **Iteratif**: Pengembangan dibagi menjadi sprint-sprint kecil (Login -> Menu -> Transaksi -> Laporan).
3.  **Efisiensi**: Fokus pada fungsionalitas utama untuk memecahkan masalah transaksi manual.
